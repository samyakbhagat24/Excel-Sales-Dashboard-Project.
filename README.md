# Excel-Sales-Dashboard-Project.
Excel Sales Performance Dashboard is an interactive Microsoft Excel project that analyzes and visualizes sales performance. It features KPI cards, Pivot Tables, Pivot Charts, and Slicers to track sales, revenue, calls handled, and AHT, enabling users to filter data and compare performance across locations, products, and months.

# Key Insights & Features
•	Built an interactive Sales Performance Dashboard in Microsoft Excel for analysing business performance.
•	Used Pivot Tables, Pivot Charts, Slicers, and KPI Cards to create dynamic and user-friendly visualizations.
•	Enabled interactive filtering by Location, Product, and Month for customized analysis.
•	Displayed key business metrics including Sales, Revenue, Calls Handled, and Average Handling Time (AHT).
•	Compared location-wise sales and revenue to identify top-performing cities and regional trends.
•	Visualized monthly product performance through comparison charts to identify seasonal patterns.
•	Supported data-driven decision-making with real-time insights and an intuitive dashboard layout
